# EduTesting

Полноценный Django-проект по теме «Веб-сервис онлайн тестирования студентов».

## Что реализовано

- регистрация и авторизация пользователей с ролями `студент` и `преподаватель`
- каталог курсов с фильтрацией по семестру и поиском
- создание курсов, тестов, вопросов, вариантов ответов и объявлений преподавателем
- расширенная карточка курса: код дисциплины, учебный год, сроки, политика оценивания
- запись студентов на курс по публичной странице или по коду доступа
- автоматическая проверка попыток, подсчёт баллов, процентов, длительности и показ подробного разбора
- преподавательская аналитика по курсам, студентам, попыткам и проходимости тестов
- студенческая панель с прогрессом по дисциплинам и лентой объявлений
- админ-панель Django
- тесты для ключевой бизнес-логики и пользовательских сценариев
- SQLite3 в качестве встроенной СУБД

## Технологии

- Python 3.12
- Django 4.2
- Django Templates
- SQLite3
- MVT (Model, View, Template)
- VS Code как рекомендованная среда разработки

## Быстрый запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py seed_demo_data
python manage.py runserver
```

После запуска приложение будет доступно по адресу `http://127.0.0.1:8000/`.

## Демо-аккаунты

- преподаватель: `teacher_demo` / `TeacherDemo123!`
- студент: `student_demo` / `StudentDemo123!`

## Проверка тестов

```bash
python manage.py test
```

## Минимальный API и Swagger

Для демонстрации серверной части в проект добавлен небольшой read-only API.

- Swagger UI: `http://127.0.0.1:8000/api/docs/`
- OpenAPI schema: `http://127.0.0.1:8000/api/schema/`
- список курсов: `http://127.0.0.1:8000/api/courses/`
- детальная информация о курсе: `http://127.0.0.1:8000/api/courses/1/`
- детальная информация о тесте: `http://127.0.0.1:8000/api/quizzes/1/`

Подробная памятка для отчёта и скриншотов находится в [docs/api_demo.md](docs/api_demo.md).

## Структура проекта

```text
student_testing/   # настройки Django-проекта
accounts/          # пользователи, роли, авторизация, профиль
testing/           # курсы, тесты, вопросы, попытки, сервис оценивания
templates/         # HTML-шаблоны
static/css/        # стили интерфейса
docs/              # пояснительная документация и аналитика
```

## Основные страницы

- `/` — главная страница сервиса
- `/courses/` — каталог курсов
- `/dashboard/` — личная панель пользователя с аналитикой
- `/accounts/signup/` и `/accounts/login/` — регистрация и вход
- `/admin/` — административный интерфейс Django

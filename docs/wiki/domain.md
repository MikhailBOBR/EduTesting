# Предметная область

## Назначение

EduTesting — веб-сервис для организации онлайн-тестирования студентов. Система решает сразу две задачи:

- помогает студенту проходить тесты, видеть результат, прогресс и рекомендации;
- даёт преподавателю инструменты управления курсом, контроля попыток и аналитики по обучению.

## Участники системы

### Студент

- записывается на курс;
- проходит тесты;
- получает результат, сравнение с прошлой попыткой и достижения;
- сохраняет черновик во время прохождения;
- отправляет апелляцию;
- получает уведомления.

### Преподаватель

- создаёт и публикует курсы, тесты, вопросы и варианты ответов;
- управляет объявлениями;
- просматривает попытки студентов;
- выставляет индивидуальные условия по тесту;
- анализирует слабые темы и студентов, требующих внимания;
- отслеживает подозрительные попытки;
- рассматривает апелляции.

## Основные проблемы, которые решает сервис

- ручная проверка тестов занимает время;
- студенту часто не хватает понятной обратной связи после попытки;
- преподавателю сложно быстро увидеть слабые темы курса;
- требуется прозрачный сценарий для повторных попыток, персональных условий и апелляций;
- при дистанционном формате важно видеть нетипичные и подозрительные попытки.

## Ключевые сценарии

### Сценарий студента

1. Войти в систему.
2. Открыть каталог курсов и записаться.
3. Запустить тест.
4. Автоматически сохранять черновик по мере прохождения.
5. Отправить попытку.
6. Получить подробный результат, сравнение с прошлой попыткой и новые достижения.

### Сценарий преподавателя

1. Создать курс и тест.
2. Опубликовать материалы.
3. Следить за попытками и уведомлениями.
4. Открыть аналитику курса и контроль подозрительных попыток.
5. При необходимости выдать индивидуальные условия.
6. Рассмотреть апелляции студентов.

## Ограничения и ожидания

- интерфейс должен оставаться простым и понятным;
- проект использует серверный рендеринг на Django Templates без отдельного SPA;
- все ключевые сценарии должны работать и через web-интерфейс, и через API;
- данные должны корректно отображаться на русском языке.

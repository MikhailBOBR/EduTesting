# Релизная проверка

## Чеклист перед релизом

- проверить миграции и запуск проекта
- убедиться, что демо-данные создаются без ошибок
- открыть web-интерфейс и пройти базовые маршруты
- открыть `/admin/` и проверить административную панель
- открыть `/api/docs/` и проверить Swagger UI
- импортировать коллекцию Postman и пройти демонстрационные запросы
- прогнать автотесты
- убедиться, что русский текст корректно отображается в шаблонах, документации и API-описании

## Соответствие ключевым требованиям

| Требование | Реализация |
| --- | --- |
| Веб-сервис онлайн-тестирования студентов | Курсы, тесты, попытки, результаты, аналитика и роли студента/преподавателя |
| Python + Django | Основа проекта и серверной логики |
| Django Templates | Клиентский слой на серверных шаблонах без отдельного SPA |
| SQLite3 | База данных в `student_testing/settings.py` |
| MVT | Явное разделение на models, views и templates |
| Межстраничная навигация | Главная, курсы, панель, курс, тест, результат, уведомления, Swagger, admin |
| Современный внешний вид | Единый базовый шаблон, стили, аккуратный Swagger UI, адаптация новых функций без перегруза |
| Серверная логика | `testing/views.py`, `testing/services.py`, `testing/analytics.py`, `testing/api_views.py` |
| Логика базы данных | `accounts/models.py`, `testing/models.py`, миграции |
| Клиентское представление | `templates/`, `static/css/` |
| Документирование API | `schema.yaml`, Swagger UI, Postman collection |
| Безопасность | Хеширование паролей, валидаторы, временная блокировка входа, secure cookies, password change |
| Администрирование | Полноценная `Django Admin` панель и демо-админ |

## Команды для финальной проверки

```bash
python manage.py migrate
python manage.py seed_demo_data
python manage.py test
python manage.py spectacular --file schema.yaml
```

## Проверка кодировки

Для защиты от проблем с русским текстом в репозиторий добавлены:

- `.editorconfig` с `charset = utf-8`
- `.gitattributes` с едиными правилами для текстовых файлов
- `<meta charset="utf-8">` в основных HTML-шаблонах

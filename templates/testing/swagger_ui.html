Быстрый старт API

1. Получите токен. 2. Нажмите Authorize. 3. Проверьте сценарий enroll → start → draft → submit.

Что показать

Сильнее всего смотрятся POST /api/auth/token/, POST /api/attempts/{id}/submit/, GET /api/my/achievements/, GET /api/courses/{id}/analytics/ и GET /api/courses/{id}/integrity/.

Рабочий режим

Фильтр, время ответа, сохранение авторизации и Try it out уже включены, поэтому Swagger подходит и для демо, и для живой проверки API.

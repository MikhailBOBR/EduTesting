Полный вход в API EduTesting: публичные обзорные методы, token auth, смена пароля, защищенный студенческий поток прохождения теста, достижения, персональные условия, апелляции, аналитика курса и преподавательский контроль подозрительных попыток.

Безопасность Token auth и смена пароля

Получите токен через POST /api/auth/token/, затем передайте Token ваш_токен в Authorize. Для защищенной смены пароля используйте POST /api/auth/password/change/.

Студент Тест, черновик и достижения

Покрыты запись на курс, старт попытки, автосохранение, отправка, просмотр результата, апелляции и личные достижения.

Преподаватель Аналитика и контроль честности

Доступны аналитика курса, список попыток, индивидуальные условия, рассмотрение апелляций и сводка по подозрительным попыткам.

Администрирование Системный контур проекта

Для веб-администрирования проекта доступна панель /admin/ с управлением пользователями, курсами, тестами, попытками, уведомлениями и безопасностью.

Быстрый старт API

1. Получите токен. 2. Нажмите Authorize. 3. Проверьте сценарий enroll → start → draft → submit.

Что показать

Сильнее всего смотрятся POST /api/auth/token/, POST /api/auth/password/change/, POST /api/attempts/{id}/submit/, GET /api/my/achievements/, GET /api/courses/{id}/analytics/ и GET /api/courses/{id}/integrity/.

Рабочий режим

Фильтр, время ответа, сохранение авторизации и Try it out уже включены, поэтому Swagger подходит и для демонстрации, и для живой проверки API.

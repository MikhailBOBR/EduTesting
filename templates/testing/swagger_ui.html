Сценарии Студент и преподаватель

Покрыты старт попытки, автосохранение, отправка, аналитика курса и просмотр завершенных попыток.

Быстрый старт API

1. Получите токен. 2. Нажмите Authorize. 3. Проверьте сценарий enroll → start → draft → submit.

Ключевые эндпоинты

Лучше всего смотрятся POST /api/auth/token/, POST /api/attempts/{id}/draft/ и GET /api/courses/{id}/analytics/.
